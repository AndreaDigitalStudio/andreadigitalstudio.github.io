## Para qué sirve

`git stash` guarda temporalmente los cambios que tienes sin confirmar, dejando tu directorio de trabajo limpio como si no hubieras tocado nada. Útil cuando necesitas cambiar de rama, hacer un `pull`, o probar otra cosa sin perder lo que estabas haciendo y sin tener que hacer un commit a medias.

## El problema que resuelve

```bash
git checkout otra-rama
```

```text
error: Your local changes to the following files would be overwritten by checkout
```

Git no te deja cambiar de rama si tienes cambios sin confirmar que entrarían en conflicto. `stash` los guarda aparte para que puedas moverte libremente.

## Comandos básicos

### Guardar los cambios actuales

```bash
git stash
```

Guarda todos los cambios (modificados y añadidos con `git add`) y vuelve el directorio de trabajo al estado del último commit.

### Ver la lista de stashes guardados

```bash
git stash list
```

```text
stash@{0}: WIP on main: a1b2c3d ajustes pendientes
stash@{1}: WIP on feature/menu: e4f5g6h trabajo en progreso
```

### Recuperar el último stash

```bash
git stash pop
```

Aplica los cambios guardados **y los elimina** de la lista de stashes.

### Recuperar sin eliminar de la lista

```bash
git stash apply
```

Útil si quieres aplicar el mismo stash en varias ramas.

### Eliminar un stash sin aplicarlo

```bash
git stash drop stash@{0}
```

### Eliminar todos los stashes

```bash
git stash clear
```

## Stash con mensaje descriptivo

Si vas a tener varios stashes, dales un nombre para identificarlos:

```bash
git stash push -m "ajustes de estilos pendientes"
```

## Incluir archivos nuevos (sin seguimiento)

Por defecto, `git stash` no guarda archivos que nunca se han añadido con `git add` (archivos nuevos). Para incluirlos también:

```bash
git stash -u
```

## Resumen

| Acción | Comando |
| --- | --- |
| Guardar cambios | `git stash` |
| Guardar con mensaje | `git stash push -m "mensaje"` |
| Incluir archivos nuevos | `git stash -u` |
| Ver lista | `git stash list` |
| Recuperar y eliminar | `git stash pop` |
| Recuperar sin eliminar | `git stash apply` |
| Eliminar uno | `git stash drop stash@{0}` |
| Eliminar todos | `git stash clear` |
