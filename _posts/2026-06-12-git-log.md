## Para qué sirve

`git log` muestra el historial de commits del repositorio: quién hizo qué cambio, cuándo, y con qué mensaje. Es la forma de entender cómo ha evolucionado el proyecto y de encontrar un commit concreto.

## Formato básico

```bash
git log
```

```text
commit a1b2c3d4e5f6...
Author: Andrea <andrea@email.com>
Date:   Wed Jun 11 18:30:00 2026 +0200

    feat: añadir página de categoría Git
```

Muestra el historial completo, del más reciente al más antiguo.

## Formato compacto — `--oneline`

```bash
git log --oneline
```

```text
a1b2c3d feat: añadir página de categoría Git
e4f5g6h fix: corregir ruta del CSS en producción
i7j8k9l docs: guía de cron y automatización
```

Cada commit en una línea: hash corto + mensaje. Ideal para ver de un vistazo el historial reciente.

## Ver ramas y fusiones — `--graph`

```bash
git log --oneline --graph --all
```

```text
* a1b2c3d (HEAD -> main) feat: añadir página de categoría Git
* e4f5g6h fix: corregir ruta del CSS
| * i7j8k9l (feature/menu) feat: menú responsive
|/
* j0k1l2m commit inicial
```

Muestra visualmente cómo se ramifican y se fusionan las ramas. Útil para entender la estructura del historial antes de hacer `merge` o `rebase`.

## Limitar resultados

```bash
git log -5
```

Muestra solo los últimos 5 commits.

```bash
git log --oneline -10
```

Combina formato compacto con límite.

## Filtrar por autor

```bash
git log --author="Andrea"
```

## Filtrar por fecha

```bash
git log --since="2026-06-01" --until="2026-06-12"
```

## Buscar por mensaje de commit

```bash
git log --grep="fix"
```

Muestra solo los commits cuyo mensaje contiene "fix".

## Ver los cambios de cada commit

```bash
git log -p
```

Muestra el log junto con el `diff` de cada commit (qué líneas cambiaron).

## Ver qué archivos cambiaron en cada commit

```bash
git log --stat
```

```text
a1b2c3d feat: añadir página de categoría Git
 git/index.html        | 12 ++++++++++++
 _layouts/git-sub.html | 25 +++++++++++++++++++++++++
 2 files changed, 37 insertions(+)
```

## Combinación útil para el día a día

```bash
git log --oneline --graph --decorate -10
```

Últimos 10 commits, en formato compacto, con la estructura de ramas y mostrando a qué ramas/etiquetas apunta cada commit (`--decorate`).

## Resumen

| Quiero... | Comando |
| --- | --- |
| Historial completo | `git log` |
| Vista compacta | `git log --oneline` |
| Ver ramas y fusiones | `git log --oneline --graph --all` |
| Limitar a N commits | `git log -N` |
| Filtrar por autor | `git log --author="nombre"` |
| Filtrar por fecha | `git log --since="fecha" --until="fecha"` |
| Buscar por mensaje | `git log --grep="texto"` |
| Ver cambios de cada commit | `git log -p` |
| Ver archivos modificados | `git log --stat` |
